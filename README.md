# Colorización de Imágenes

## Descripción

Proyecto de colorización de imágenes automática de imágenes mediante RBF y CNN.
Se presenta una comparativa entre diferentes estructuras para la colorización de imágenes de carícaturas 

## Dataset
El dataset utilizado consta de 10 diferentes clases, las cuales son 10 diferentes personajes de caricaturas populares. Este se encuentra dividido en un entrenamiento con 10k imágenes de cada clase, y 2k para pruebas
https://www.kaggle.com/datasets/volkandl/cartoon-classification
